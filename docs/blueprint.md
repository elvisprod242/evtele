# **App Name**: StreamVerse

## Core Features:

- M3U Playlist Parsing: Parse and interpret M3U playlist files to extract channel information.
- Channel Listing & Guide: Display a list of available TV channels with an electronic program guide (EPG).
- Video Playback: Integrate a video player capable of streaming content from the parsed M3U URLs.
- Channel Sorting & Favorites: Allow users to sort channels and mark channels as favorites.
- Recommendation Engine: AI tool provides personalized channel recommendations based on viewing history.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082), evocative of late-night movie watching.
- Background color: Very dark gray (#1A1A1A) for a modern dark theme.
- Accent color: Vibrant magenta (#FF00FF) for highlights and selections, creating a dynamic contrast.
- Body and headline font: 'Inter' for a clean, modern, readable interface.
- Use simple, recognizable icons for navigation and channel categories.
- Grid-based layout for channel listings to maximize screen space.
- Smooth transitions and loading animations to enhance the viewing experience.
