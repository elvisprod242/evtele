# StreamVerse

This is a Next.js application for a streaming TV channel app that can parse M3U playlists, display a channel guide, and play video streams.

## Features

- M3U Playlist Parsing
- Channel Listing & EPG
- Video Playback
- Favorites & Sorting
- AI-powered Recommendations

To get started, run `npm run dev` and open http://localhost:9002 in your browser.
